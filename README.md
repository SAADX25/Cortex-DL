# 🧠 Cortex DL 
> The Ultimate Modern Video & Audio Downloader. 
  
**Cortex DL** is a powerful, sleek, and user-friendly desktop application designed to download high-quality video and audio from popular platforms like YouTube, Facebook, Instagram, and TikTok. Built with performance and aesthetics in mind. 
  
## ✨ Key Features 
* **🚀 Smart Downloads:** Auto-detects video/audio formats with high-speed processing. 
* **🎨 Modern Dark UI:** A beautiful, eye-friendly dark theme with smooth animations. 
* **🌍 Multi-Language Support:** Full support for **Arabic (RTL)** and English. 
* **📋 Clipboard Monitor:** Automatically detects copied links and prepares them for download. 
* **⚡ Quick Access:** Built-in shortcuts to browse top video sites instantly. 
* **🔒 Privacy Focused:** Runs locally on your machine with no tracking. 
  
## 🛠️ Tech Stack 
* **Framework:** Electron & React 
* **Styling:** Tailwind CSS 
* **Language:** TypeScript 
* **Build Tool:** Electron Builder
